## 🚀 طريقة النشر

### الطريقة السريعة (GitHub Pages)

1. **أنشئ مستودعاً جديداً** على GitHub (عاماً Public)
2. **ارفع الملفات** كلها (بما فيها `index.html`)
3. اذهب إلى **Settings → Pages**
4. اختر **Source: GitHub Actions**
5. الموقع سيكون جاهزاً خلال دقيقة!

### رفع الملفات عبر سطر الأوامر

```bash
git init
git add .
git commit -m "🚀 أول نشر — محاكاة توازن جسم صلب"
git branch -M main
git remote add origin https://github.com/<اسم_المستخدم>/<اسم_المستودع>.git
git push -u origin main
```
